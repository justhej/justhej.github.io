Swp_1.1: Did you say noob?

2023-12-23
1 minute read

This project is unique in that I’ve decided to re-solve the Swift Playgrounds puzzles.

I say ’re-solve’ because I did them a long time ago as a noob, when I used to find it overwhelming.

This time around, with more programming muscle under my belt (and knowing what pre-defined data structures are available for use), I will showcase my creativity and problem-solving within the constraints set by Swift Playgrounds; I will list the noob version first, followed by several solutions that progressively get better–culminating in “hej-level” solutions that I think can’t be bested.

In the title, 1.1 means for book 1, which is called “Let’s Code 1” and in that book, chapter 1. Chapter 1 consists of 7 puzzles. In my solution repo (see below), I list the solutions by puzzle names and chapter numbers like so:

1.1.1: Issuing Commands

The predefined functions are introduced and with it a simple solution is invoked:

1
2moveForward()
3moveForward()
4moveForward()
5collectGem()

At first glance, the only other sensible thing that can be done is:

1func move(dist: Int) {
2    for i in 1...dist{
3        moveForward()
4    }
5}
6
7move(dist:3)
8collectGem()

Which is more lines than the original ’noob’ solution.

Upon reflection, why not use a while loop?

1while !isBlocked{
2    moveForward()
3    collectGem()
4}

That’s less lines than the noob version and doesn’t involve repitition bloat.

The final challenge in chapter 1 expects something like this (which is the solution I came up with when I did this for the first time as a noob:

1moveForward()
2moveForward()
3moveForward()
4collectGem()
5moveForward()
6moveForward()
7moveForward()
8moveForward()
9toggleSwitch()

Several months later, I came up with this:

 1
 2while !isOnClosedSwitch{
 3    moveForward()
 4    if isOnGem{
 5        collectGem()
 6    }
 7    if isOnClosedSwitch{
 8        toggleSwitch()
 9        break
10    }
11}

Not satisfied, I tweaked it further:

1while !isOnClosedSwitch {
2    moveForward()
3    isOnGem ? collectGem() : ()
4}
5
6    isOnClosedSwitch ? toggleSwitch() : ()
